# map の比較

まずは単純な map 関数について、for 文との違いを見ていく。配列の各要素を 2 倍する処理を考えてみよう。

まずは for 文

```js
const array = [1, 2, 3];
const results = [];
for (item of array) {
  results.push(item * 2);
}
console.log(results);
//=> [2, 4, 6]
```
